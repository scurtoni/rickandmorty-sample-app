# rickandmorty-sample-app

Simple React App listing Rick and Morty characters.
Show the list of Rick and Morty character and show location and episodes on click to "details" button.

Core:
React + Typescript

Linter:
ESLint
Husky

Code formatter:
Prettier

State Management:
Redux
Redux Toolkit
Redux Saga

Testing:
React Testing Library
Jest
